MICROSERVICES INTEGRATION GUIDE
================================

This guide explains how to integrate the microservices with the main backend API.

## Quick Start

### 1. Import the gRPC Client Manager

```python
from backend.microservices.common.grpc_clients import grpc_clients
```

### 2. Use in Your Backend APIs

#### Example: Integrating Route Service in Search API

```python
# backend/api/search.py
from fastapi import APIRouter, Depends
from backend.microservices.common.grpc_clients import grpc_clients
from backend.microservices.route_service.src import route_pb2
from google.protobuf.timestamp_pb2 import Timestamp
from datetime import datetime

router = APIRouter()

@router.post("/search-routes")
async def search_routes(request: SearchRequest):
    """Search routes using gRPC microservice"""
    
    # Get client
    client = grpc_clients.get_route_client()
    
    # Create proto request
    dep_time = Timestamp()
    dep_time.FromDatetime(request.departure_date)
    
    grpc_request = route_pb2.RouteRequest(
        from_station_id=str(request.from_station_id),
        to_station_id=str(request.to_station_id),
        departure_date=dep_time,
        max_transfers=request.max_transfers or 3,
        num_passengers=request.num_passengers or 1,
        num_alternatives=request.num_alternatives or 5
    )
    
    try:
        # Call gRPC service
        response = await client.FindRoutes(grpc_request)
        
        # Convert proto response to JSON
        routes = [
            {
                "route_id": route.route_id,
                "legs": [
                    {
                        "train_number": leg.train_number,
                        "departure": leg.departure_time.ToJsonString(),
                        "arrival": leg.arrival_time.ToJsonString(),
                        "duration": leg.duration_mins,
                    }
                    for leg in route.legs
                ],
                "total_duration": route.total_duration_mins,
                "total_price": route.total_price,
                "reliability": route.reliability_score,
            }
            for route in response.routes
        ]
        
        return {
            "success": True,
            "search_id": response.search_id,
            "routes": routes,
            "latency_ms": response.latency_ms,
        }
    
    except Exception as e:
        logger.error(f"Route search failed: {e}")
        return {
            "success": False,
            "error": str(e),
        }
```

@router.get("/check-availability")
async def check_availability(train_id: str, date: str, num_passengers: int = 1):

#### Example: Integrating Booking Service

```python
# backend/api/bookings.py

@router.post("/create-booking")
async def create_booking(request: BookingRequest):
    """Create booking using gRPC microservice"""
    
    client = grpc_clients.get_booking_client()
    
    travel_date = Timestamp()
    travel_date.FromDatetime(request.travel_date)
    
    passengers = [
        booking_pb2.Passenger(
            name=p.name,
            age=p.age,
            gender=p.gender,
            berth_preference=p.berth_preference,
        )
        for p in request.passengers
    ]
    
    grpc_request = booking_pb2.BookingRequest(
        user_id=request.user_id,
        trip_id=request.trip_id,
        from_stop_id=request.from_stop_id,
        to_stop_id=request.to_stop_id,
        travel_date=travel_date,
        quota_type=request.quota_type,
        passengers=passengers,
        payment_method="razorpay",
        total_amount=request.total_amount,
    )
    
    response = await client.InitiateBooking(grpc_request)
    
    return {
        "success": response.success,
        "pnr": response.pnr,
        "booking_id": response.booking_id,
        "status": response.status,
        "total_amount": response.total_amount,
    }
```

## Configuration

Set environment variables for microservice endpoints:

```bash
# .env or environment
ROUTE_SERVICE_HOST=localhost
ROUTE_SERVICE_PORT=50051
ROUTE_SERVICE_TIMEOUT=30

INVENTORY_SERVICE_HOST=localhost
INVENTORY_SERVICE_PORT=50052
INVENTORY_SERVICE_TIMEOUT=30

BOOKING_SERVICE_HOST=localhost
BOOKING_SERVICE_PORT=50053
BOOKING_SERVICE_TIMEOUT=60
```

Load in your config:

```python
# backend/config.py
import os

ROUTE_SERVICE_HOST = os.getenv("ROUTE_SERVICE_HOST", "localhost")
ROUTE_SERVICE_PORT = int(os.getenv("ROUTE_SERVICE_PORT", 50051))
ROUTE_SERVICE_TIMEOUT = int(os.getenv("ROUTE_SERVICE_TIMEOUT", 30))

# ... similar for other services
```

Then use in client creation:

```python
from backend.config import ROUTE_SERVICE_HOST, ROUTE_SERVICE_PORT, ROUTE_SERVICE_TIMEOUT

route_client = grpc_clients.get_route_client(
    host=ROUTE_SERVICE_HOST,
    port=ROUTE_SERVICE_PORT,
    timeout_seconds=ROUTE_SERVICE_TIMEOUT
)
```

## Error Handling

Always handle gRPC exceptions:

```python
import grpc

try:
    response = await client.FindRoutes(request)
except grpc.RpcError as e:
    logger.error(f"gRPC error: {e.code()}, {e.details()}")
    # Handle specific error codes
    if e.code() == grpc.StatusCode.DEADLINE_EXCEEDED:
        return {"error": "Request timeout"}
    elif e.code() == grpc.StatusCode.UNAVAILABLE:
        return {"error": "Service unavailable"}
    else:
        return {"error": "Service error"}
except Exception as e:
    logger.error(f"Unexpected error: {e}")
    return {"error": "Internal error"}
```

## Best Practices

1. **Use Timeouts**: Always set appropriate timeouts for gRPC calls
2. **Handle Errors**: Implement proper error handling and retry logic
3. **Log Calls**: Log all gRPC calls for debugging
4. **Monitor Health**: Check service health regularly
5. **Cache Results**: Cache frequently requested data
6. **Circuit Breaker**: Let circuit breaker handle failures automatically
7. **Async/Await**: Use async calls for better performance
8. **Pool Connections**: The client manager reuses connections

DEPLOYMENT INSTRUCTIONS
=======================

STEP 1: Install Dependencies
   cd routemaster_agent
   pip install -r requirements.txt
   playwright install chromium

STEP 2: Set Environment Variables
   export GEMINI_API_KEY="your-key-from-ai.google.dev"
   export DATABASE_URL="postgresql://user:pass@host/db"
   export RMA_DATABASE_URL="postgresql://user:pass@host/rma"

STEP 3: Verify Installation
   python deployment_verify.py
   # Should show: STATUS: ALL FILES VERIFIED - READY FOR DEPLOYMENT

STEP 4: Update main.py
   Add to routemaster_agent/main.py:
   
   from routemaster_agent.manager import RouteMasterAgentManager
   from routemaster_agent.api_dashboard import router as dashboard_router
   from routemaster_agent.core.reasoning_loop import ReasoningLoop
   
   app.include_router(dashboard_router)
   
   manager = None
   
   @app.on_event("startup")
   async def startup_event():
       global manager
       reasoning_loop = ReasoningLoop(gemini_client=gemini_client)
       manager = RouteMasterAgentManager(reasoning_loop, SessionLocal())

STEP 5: Update scheduler.py
   Wire manager methods to scheduled tasks:
   
   result = await manager.collect_monthly_schedule_updates()
   result = await manager.collect_daily_live_status()

STEP 6: Test Endpoints
   # Test metrics
   curl -X GET http://localhost:8000/api/agent/dashboard/metrics
   
   # Test unlock route
   curl -X POST http://localhost:8000/api/agent/unlock-route-details \
     -H "Content-Type: application/json" \
     -d '{"train_number":"12951","source":"NDLS","dest":"CNB","date":"2026-02-17"}'
   
   # Test WebSocket
   wscat -c ws://localhost:8000/api/agent/ws/dashboard

STEP 7: Start Server
   uvicorn routemaster_agent.main:app --host 0.0.0.0 --port 8000

STEP 8: Setup Grafana Dashboard
   - Add data source: http://localhost:8000
   - Create panels for metrics
   - Add command buttons
   - Setup WebSocket real-time updates

STEP 9: Deploy to Production
   - Test in staging environment
   - Monitor logs and metrics
   - Deploy with confidence!
